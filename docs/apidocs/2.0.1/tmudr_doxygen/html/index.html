C++ Interface for Trafodion UDRs

2.0.1

Class tmudr::UDR contains the code written by the UDR writer as methods. It also contains default implementations for methods the UDR writer chooses not to provide. Start with this class if you want to explore the interface.

Classes tmudr::UDRInvocationInfo and tmudr::UDRPlanInfo describe a specific invocation of a UDR, e.g. number, types and values of parameters, UDR name, etc.

For an introduction, see https://cwiki.apache.org/confluence/display/TRAFODION/Tutorial%3A+The+object-oriented+UDF+interface
